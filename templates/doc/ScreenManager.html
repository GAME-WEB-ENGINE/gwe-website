ScreenManager()

Singleton représentant un gestionnaire d'écrans.

Constructor

new ScreenManager()

Créer un gestionnaire d'écrans.

Methods

draw(viewIndex)

Répartition du rafraichissement graphique à tous les écrans. Nota bene: Si un écran est bloquant, celui-ci va stopper la répartition descendante.
Parameters:
Name Type Description
viewIndex number Index de la vue en cours.

handleEvent(event)

Répartition des entrées utilisateur à tous les écrans. Nota bene: Si un écran est bloquant, celui-ci va stopper la répartition descendante.
Parameters:
Name Type Description
event InputEvent Evènement d'entrée utilisateur.

requestPopScreen()

Commande la suppression du dernier écran.

requestPushScreen(newTopScreen, args)

Commande l'ajout d'un écran.
Parameters:
Name Type Description
newTopScreen Screen Ecran à ajouter.
args object Données transitoires.

requestSetScreen(newScreen, args)

Commande l'ajout d'un écran et supprime tous les écrans courants.
Parameters:
Name Type Description
newScreen Screen Ecran à ajouter.
args object Données transitoires.

update(ts)

Répartition de la mise à jour à tous les écrans. Nota bene: Si un écran est bloquant, celui-ci va stopper la répartition descendante.
Parameters:
Name Type Description
ts number Temps passé depuis la dernière mise à jour.
