ScriptMachine()

Classe représentant une machine de script. Cette classe crée un contexte d'exécution et permet de lancer un script (fichier JSC). Note: Chaque script à sa propre machine d'exécution.

Constructor

new ScriptMachine()

Créer une machine de script.
