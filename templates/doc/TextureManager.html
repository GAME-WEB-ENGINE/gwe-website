TextureManager()

Singleton représentant un gestionnaire de ressources texture.

Constructor

new TextureManager()

Créer un gestionnaire de ressources texture.

Methods

deleteTexture(path)

Supprime la ressource.

Name Type Description

getTexture(path) → {Texture}

Récupère la ressource. Si celle-ci n'est pas pré-chargée c'est alors la texture par default qui est retournée.
Parameters:
Name Type Description
path string Le chemin du fichier de texture.

Parameters:
Name Type Description
path string Le chemin du fichier texture.
