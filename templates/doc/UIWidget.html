UIWidget()

Classe représentant un élément d'interface utilisateur.

Constructor

new UIWidget()

Créer un élément d'interface utilisateur.

Methods

animate(animation)

Applique une animation css.
Parameters:
Name Type Description
animation string Description de l'animation.

delete()

Destructeur. Nota bene: Entraine la désinscription des évènements utilisateur et détache le noeud HTML de son parent.

focus()

Donne le focus. Nota bene: Souscription aux évènements utilisateur et ajout de la classe 'u-focused'.

getId() → {string}

Retourne l'identifiant.

hide()

Rends le widget invisible.

isEnabled() → {boolean}

Vérifie si le widget est activé.

isFocused() → {boolean}

Vérifie si le widget est focus.

isSelected() → {boolean}

Vérifie si le widget est selectionné.

isShow() → {boolean}

Vérifie si le widget est visible.

onKeyDown()

Fonction appelée lorsque le widget est "focus" et qu'une touche est appuyée.

onKeyDownOnce()

Fonction appelée lorsque le widget est "focus" et qu'une touche est appuyée ponctuellement.

setEnabled(enabled)

Définit l'activation. Nota bene: Ajoute la classe 'u-disabled'.
Parameters:
Name Type Description
enabled boolean Si vrai, le widget est activé.

setId(id)

Définit l'identifiant.
Parameters:
Name Type Description
id string L'Identifiant.

setSelected(selected)

Définit la selection. Nota bene: Ajoute la classe 'u-selected'.
Parameters:
Name Type Description
selected boolean Si vrai, le widget est selectionné.

show()

Rends le widget visible.

unfocus()

Enlève le focus. Nota bene: Désinscription aux évènements utilisateur et suppréssion de la classe 'u-focused'.

update(ts)

Fonction de mise à jour.
Parameters:
Name Type Description
ts number Temps passé depuis la dernière mise à jour.
